## While Loop

The ```while``` loop executes a block of code as long as the specified condition is true.

### Syntax
```
while (true) {
    #code1;
}
```
### Example1
A program to print number from 1 to 10 using while loop.

{% highlight php %}
<?php 
$x = 1; 

while($x <= 10) {
    $x++;
} 
?>
{% endhighlight %}
